# Contador de Números Primos em Perl

Computação numérica de alto desempenho em Perl, incluindo contagem otimizada de números primos.

## Recursos

- Soma de quadrados usando fórmula O(1)
- Contagem rápida de números primos usando biblioteca otimizada
- Cálculo de números complexos

## Requisitos

- Perl
- Math::Prime::Util

## Instalação

### Debian / Ubuntu

```bash
$ sudo apt install libmath-prime-util-perl
```

## Uso

```bash
$ perl script.pl
```

## Exemplo de saída

```bash
Sum of squares: 333333833333500000
Prime count: 78498
Complex result: -7+24i
```
